1. Классы и структуры, отличия. 
Класс - ссылочный тип, структура - значимый. ВАжное отличие ссыл. типа от типа знач. это то, что структура
не может быть null. Так как структура не может быть null, то она должн абыть проинициализирована. При этом инициализация
структуры происходить по умолчанию нулевыми значениями. Вследствие мы не можем создавать конструктор по умолчанию для структур.
Структуры имеют модиф доступа: public, internal, private.Структуры нельзя наследовать 

2.Чем отличаются константы и поля, доступные только для чтения
Особенностью констант является то, что их значение можно установить только один раз.
Поля для чтения можно инициализировать при их объявлении либо на уровне класса, либо инициилизировать и изменять в конструкторе.
Инициализировать или изменять их значение в других местах нельзя, можно только считывать их значение.

3.Когда использовать StringBuilder предпочтительнее, чем string
Использовать класс String лучше в следующих случаях:
*При небольшом количестве операций и изменений над строками
*При выполнении фиксированного количества операций объединения.
*Когда надо выполнять масштабные операции поиска при построении строки, например IndexOf или StartsWith.
Класс StringBuilder не имеет подобных методов.

Класс StringBuilder рекомендуется использовать в следующих случаях:
*При неизвестном количестве операций и изменений над строками во время выполнения программы
*Когда предполагается, что приложению придется сделать множество подобных операций

6. В чем различие между ключевыми словами "ref" и "out"?
ref сообщает компилятору, что объект инициализирован перед входом в функцию,
 а out сообщает компилятору, что объект будет инициализирован внутри функции.

7. Конструктор, виды конструкторов
Конструкторы - специальные методы. Конструкторы вызываются при создании нового объекта данного класса.
Конструкторы выполняют инициализацию объекта.
Если в классе не определено ни одного конструктора, то для этого класса автоматически создается конструктор без параметров.
Конструкторы бывают с параметрами и без параметров.
